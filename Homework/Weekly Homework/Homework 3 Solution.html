On eClass you will find a data set called MaleBody.csv. It contains the physical measurements of 252 males (Homo sapiens). Evaluate whether the percentage body fat using Brozek’s equation (457/Density - 414.2), which is given by the column ‘body.fat’, is normally distrubuted.

maleBod <- read.csv("../Data/MaleBody.csv")

ggplot(maleBod, aes(sample = body.fat)) +
    stat_qq() +
    stat_qq_line()

There is some deviation in the tails, but no obvious systematic curving of the points. So we are probably safe in assuming normality. Note that it’s okay if the line the points form don’t angle perfectly with the solid line. What matters most is that the points themselves form a relatively straight line with no obvious curvature.

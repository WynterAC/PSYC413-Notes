Question 2:

Using α=0.05α=0.05, assume the data come from a normally distributed population and test the hypothesis that sexual ads are remembered better than the nonsexual ads. i.e., H0:μsexy−μnotSexy>0H0:μsexy−μnotSexy>0. Use of t.test() is not permitted.

Make sure to report:

- the test statistic

- the degrees of freedom

- p-value

- Whether you reject or fail to reject H0H0

#Calculate Difference Scores
diffScores <- adData$AdRecall[adData$Image == "Sexy"] -
                adData$AdRecall[adData$Image == "NotSexy"]

#Summary Stats
N <- length(diffScores)
M <- mean(diffScores)
mu <- 0
se <- sd(diffScores)/sqrt(N)

#Test Statistic
t <- (M - mu)/se

# Degrees of Freedom
df <- N - 1

# p-val
#H0: sexy > nonSexy 
# i.e., sexy - nonSexy > 0
p <- pt(t, df = df)
## [1] "T = 0.859"
## [1] "df = 38"
## [1] "p = 0.802"
## [1] "We fail to reject the null hypothesis that H0: Sexy - NotSexy > 0"

You own an ice cream business and you would like to create a linear model that could predict the daily revenue in dollars based on the outside air temperature (degrees celsius). Use the data IceCreamData.csv to create this model.

Use of lm() and cor is not permitted. Report the model’s formula.

iceCream <- read.csv("../Data/IceCreamData.csv", header = TRUE)

#Sample size
N <- nrow(iceCream)
#Correlation
Temp_Z = (iceCream$Temperature - mean(iceCream$Temperature)) /
            sd(iceCream$Temperature)
Rev_Z = (iceCream$Revenue - mean(iceCream$Revenue)) /
            sd(iceCream$Revenue)

R = sum(Temp_Z * Rev_Z)/(N - 1)
#Slope
b1 <- R * (sd(iceCream$Revenue) / sd(iceCream$Temperature))

#Intercept
b0 <- mean(iceCream$Revenue) - b1 * mean(iceCream$Temperature)

#Formula
paste("Y = ", round(b0, 3), " + ", round(b1, 3), " X", sep = "")
## [1] "Y = 44.831 + 21.444 X"
